# Blog-Website

A fully functional blog website built with HTML, CSS, and JavaScript.

## Features

- **4 Blog Posts** across Technology, Travel, Science, and Lifestyle categories — each with a title, publication date, placeholder image, and multi-paragraph content
- **Comment Section** on every post — users can submit their name and a message; comments are saved in `localStorage` so they persist across page reloads
- **Category Navigation** — filter posts by category using the top navigation bar
- **Live Search** — find posts by keyword; searches across titles, content, and tags
- **Social Sharing Buttons** — Twitter/X, Facebook, and LinkedIn share links on every post
- **Tags** — each post is labelled with relevant topic tags
- **Responsive Design** — mobile-friendly layout with a hamburger menu and CSS media queries for tablet and mobile screens

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main HTML file — structure and content |
| `styles.css` | Stylesheet — layout, colours, typography, and media queries |
| `script.js` | JavaScript — navigation, search, filtering, and comment handling |

## Getting Started

Open `index.html` directly in any modern browser — no build step or server required.
